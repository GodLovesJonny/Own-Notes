# Bits, Bytes and Integer

只有ENIAC使用十进制进行算术运算    
    
二进制 模拟信号 设定阈值可以起到类似降噪抗干扰的作用    

隐式表示数值集合 如：01000001 {0, 6}    
> & -> Intersection
> | -> Union
> ^ -> Symmetric difference
> ~ -> Complement
    
**&&, ||, !**    
+ View 0 as "false"
+ Anything nonzero as "true"
+ Always return 0 or 1
+ **Early termination**
